HTML feature

  1. Canvas: A tag that renders images into an HTML document
  2. New doctype: Indicates that the page is HTML
  3. Web Storage: Includes session storage and local storage
  4. Geolocation: Allows developers to create rich internet applications without third party APIs and browser plug-ins
  5. HTML5 drag: Includes native audio and video support without Flash
  6. Editable content: Allows visitors to edit the content of the HTML control
  7. Video support: Allows for the playback of both audio and video files in browsers
