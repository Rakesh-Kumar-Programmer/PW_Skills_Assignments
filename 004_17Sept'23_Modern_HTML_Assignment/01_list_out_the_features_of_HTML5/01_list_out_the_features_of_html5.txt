HTML5 is a standard for presenting and structuring content on the World Wide Web. It's a collaboration between the World Wide Web Consortium (W3C) and the Web Hypertext Application Technology Working Group.

some features of HTML5:

-Canvas: A tag that renders images into an HTML document 
-New doctype: Indicates that the page is HTML 
-Web Storage: Includes session storage and local storage 
-Geolocation: Allows developers to create rich internet applications without third party APIs and browser plug-ins 
-HTML5 drag: Includes native audio and video support without Flash 
-Editable content: Allows visitors to edit the content of the HTML control 
-Video support: Allows for the playback of both audio and video files in browsers
