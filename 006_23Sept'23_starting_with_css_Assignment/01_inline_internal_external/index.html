Question: Create a simple page with some div tags and show different ways to add CSS as well as what happens when you target the same elements with inline, internal, and external CSS. Also, utilize comments in the project where required.

Answer: There are three ways to add CSS. via external file . Via internal style tags in head part of the page and as inline CSS. Out of these three inline css has highest priority. I am using inline css (text-align and color) in h1 tag and paragraph tag and I am using internal css (display:flex,border,margin,padding,background-color) pointing to div tag whose class is flex-container. I am importing external css via link tag in head section and change background-color & color by class selector in city1,city2,city3 .
